### Project description

The customers of this project are IT-Agency and beauty online store [New Beauty Box](https://newbeautybox.ru/) 

The main goals of this project are:

1. To link the orders to the source, implementing an SQL-algorithm
2. To build dashboards on the received data in Power BI for three stakeholders: advertising vendor, marketing director and CEO.
