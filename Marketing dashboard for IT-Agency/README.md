### Project description

The customers of this project are IT-Agency and beauty online store [New Beauty Box](https://newbeautybox.ru/) 

The main goals of this project are:

1. To link the orders to the source, implementing an SQL scripts.
2. To build dashboards on the received data in Power BI for three stakeholders: advertising vendor, marketing director and CEO.

#### Dashbord for marketing director

The dashboard for the marketing director is the most detailed in data. It represents key marketing information:

* distribution of orders status and last-click attribution;
* conversions to payment and orderd cancellation;
* revenue, spends, profit and ROMI dynamics;
* average check, customer acqusition cost (CAC).
